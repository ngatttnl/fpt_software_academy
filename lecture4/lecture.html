Responsive

Một số điều cần nhớ:

  • Điều chỉnh giao diện trên các thiết bị khác nhau bằng cách sử dụng các media queries và các breakpoint.
  • Chọn kích thước và cách sắp xếp nội dung cho phù hợp với màn hình, bằng cách sử dụng các đơn vị đo linh hoạt như %, vw, vh, em, rem.
  • Sử dụng một lưới thiết kế (grid system) để giúp sắp xếp các phần tử trên trang web theo một cách đồng nhất và thuận tiện.
  • Tối ưu hóa hình ảnh và đa phương tiện để giảm thời gian tải trang và tối ưu hóa hiển thị trên các thiết bị khác nhau.
  • Đảm bảo rằng trang web của bạn còn hoạt động tốt khi được truy cập từ các thiết bị có kích thước khác nhau, bao gồm cả màn hình lớn, máy tính bảng, điện thoại di động và các thiết bị khác.
  • Kiểm thử (test) và sửa lỗi để đảm bảo rằng trang web của bạn hoạt động tốt trên tất cả các thiết bị và trình duyệt.

Media queries

Media queries là một công cụ mạnh mẽ trong CSS để tạo ra thiết kế responsive. Nó cho phép các trang web hiển thị khác nhau tùy thuộc vào kích thước màn hình hoặc thiết bị của người dùng. Khi sử dụng media queries, ta có thể xác định các phương tiện hiển thị cụ thể cho mỗi kích thước màn hình khác nhau.
Media queries được sử dụng bằng cách sử dụng từ khóa @media trong CSS. Cú pháp chung của một media query như sau: @media (media feature) { /* CSS styles */ } Trong đó, media feature là một số tính năng mà chúng ta sử dụng để xác định các điều kiện cho media query. Ví dụ, các media feature thông thường bao gồm:

  • min-width: kích thước màn hình tối thiểu
  • max-width: kích thước màn hình tối đa
  • orientation: hướng của thiết bị
  • aspect-ratio: tỷ lệ khung hình của màn hình
  • resolution: độ phân giải của màn hình

Chúng ta có thể sử dụng bất kỳ số lượng media feature nào và kết hợp chúng với nhau để tạo ra các điều kiện cho media query.
Ngoài ra, media queries có thể được sử dụng để tùy chỉnh các thuộc tính CSS, chẳng hạn như màu sắc, kích thước, khoảng cách, v.v. tùy theo kích thước màn hình. Bằng cách này, chúng ta có thể cải thiện trải nghiệm người dùng trên các thiết bị khác nhau mà không cần tạo nhiều phiên bản của cùng một trang web.

Ví dụ, đoạn CSS sau sẽ thay đổi font-size của đoạn văn bản khi chiều rộng của màn hình nhỏ hơn hoặc bằng 600px: 
@media (max-width: 600px) {
  p {
    font-size: 14px;
  }
}

Khi kích thước màn hình đạt đến 600px hoặc nhỏ hơn, font-size của đoạn văn bản sẽ được thay đổi thành 14px.

Một số ví dụ dùng media queries

  1. Sử dụng media query với not để áp dụng kiểu CSS chỉ cho các màn hình lớn hơn 768px: arduino 
    @media not screen and (max-width: 768px) {
  /* CSS styles for screens larger than 768px */
}
    @media not screen và (max-width: 768px) sẽ áp dụng các kiểu cho màn hình không phải là màn hình và có chiều rộng lớn hơn 768 pixel.
    Nói cách khác, mã này sẽ áp dụng kiểu cho các thiết bị như máy in, máy chiếu hoặc các thiết bị không có màn hình (not screen), và cho màn hình có chiều rộng lớn hơn 768 pixel.
  2. Sử dụng media query với only để áp dụng kiểu CSS chỉ cho các thiết bị iPhone 6, 7 và 8: 
    @media only screen and (device-width: 375px) and (device-height: 667px) and (-webkit-device-pixel-ratio: 2) {
  /* CSS styles for iPhone 6, 7, and 8 */
}
  3. Sử dụng media query với print để áp dụng kiểu CSS chỉ cho trang in ấn: 
    @media print {
  /* CSS styles for printing */
}
  4. Sử dụng media query để áp dụng kiểu CSS chỉ cho các màn hình có độ phân giải cao hơn 300dpi: 
    @media screen and (min-resolution: 300dpi) {
  /* CSS styles for high-resolution screens */
}
  5. Sử dụng media query để áp dụng kiểu CSS chỉ cho các màn hình hiển thị ở chế độ ngang: 
    @media screen and (orientation: landscape) {
  /* CSS styles for landscape screens */
}
