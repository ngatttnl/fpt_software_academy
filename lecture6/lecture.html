Flexbox

Flexbox là một công nghệ trong CSS, giúp cho việc thiết kế giao diện web trở nên linh hoạt và dễ dàng hơn. Nó cho phép bạn xác định các item trong container của bạn sẽ được sắp xếp như thế nào, độ rộng của chúng, khoảng cách giữa chúng, cách thức chúng tự động điều chỉnh kích thước theo container, và nhiều tính năng khác nữa.

Một số điều cần nhớ:

  1. Flexbox sử dụng một container để bao bọc các item. Các thuộc tính Flexbox được áp dụng cho container. Các item được xếp theo hướng ngang hoặc dọc trong container sử dụng thuộc tính flex-direction. 
            .container {
          display: flex;
          flex-direction: row; /* hoặc column */
        }
  2. Các item có thể căn chỉnh theo hướng chính giữa hoặc hướng phụ bằng thuộc tính justify-content và align-items. 
    .container {
          display: flex;
          justify-content: center; /* căn giữa theo chiều ngang */
          align-items: center; /* căn giữa theo chiều dọc */
        }
  3. Các item có thể đổi vị trí với nhau bằng thuộc tính order. 
            .item {
          order: 2; /* item này sẽ được đưa lên sau item có order là 1 */
        }
  4. Các item có thể được co hoặc giãn ra bằng thuộc tính flex-grow và flex-shrink. 
            .item {
          flex-grow: 1; /* item sẽ giãn ra để đầy không gian còn trống trong container */
          flex-shrink: 0; /* item sẽ không bị co lại khi không đủ không gian */
        }
  5. Các item có thể được sắp xếp theo thứ tự ưu tiên bằng thuộc tính flex-basis. 
            .item {
          flex-basis: 50%; /* item sẽ được cấp đủ không gian 50% trong container */
        }
