Grid layout

Grid layout là một phương pháp sắp xếp các phần tử trong CSS dựa trên lưới (grid) với các hàng (rows) và cột (columns). Đây là một cách tiện lợi để sắp xếp layout, đặc biệt là với những trang web có nội dung phức tạp và yêu cầu tương thích với nhiều thiết bị khác nhau. Dưới đây là một số vấn đề cần nhớ khi sử dụng grid layout trong CSS:

  1. Định nghĩa lưới (grid) bằng cách sử dụng thuộc tính display: grid trên phần tử cha của các phần tử cần sắp xếp.
  2. Để xác định số lượng cột và hàng trong lưới, sử dụng các thuộc tính grid-template-columns và grid-template-rows.
  3. Để xác định kích thước của một ô trong lưới, sử dụng thuộc tính grid-template-areas và định nghĩa các vùng trong lưới.
  4. Sử dụng thuộc tính grid-column và grid-row để chỉ định vị trí của các phần tử trong lưới.
  5. Có thể sử dụng các giá trị như auto, min-content, max-content, fit-content và fr để xác định kích thước và vị trí của các phần tử trong lưới.
  6. Sử dụng các thuộc tính như grid-gap và grid-row-gap để xác định khoảng cách giữa các ô trong lưới.
Grid layout có tính năng mạnh mẽ, nhưng cũng có thể gây khó khăn trong việc tối ưu hóa và bảo trì layout trên nhiều thiết bị khác nhau, đặc biệt là với các trình duyệt cũ.

Ví dụ

Xác định số cột và hàng trong lưới:

  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* xác định 3 cột bằng nhau */
    grid-template-rows: repeat(4, 100px); /* xác định 4 hàng với chiều cao là 100px */
  }

Xác định kích thước của từng ô trong lưới bằng grid-template-areas:

  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(4, 100px);
    grid-template-areas:
      "header header header"
      "sidebar content content"
      "sidebar content content"
      "footer footer footer";
  }
  
  .header {
    grid-area: header;
  }
  
  .sidebar {
    grid-area: sidebar;
  }
  
  .content {
    grid-area: content;
  }
  
  .footer {
    grid-area: footer;
  }
Trong ví dụ này, lưới bao gồm 4 hàng và 3 cột, và được chia thành 4 khu vực với các kích thước và vị trí khác nhau. Các phần tử HTML được xác định sử dụng thuộc tính grid-area.

Xác định kích thước và vị trí của từng ô trong lưới bằng grid-column và grid-row:

  .container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(4, 100px);
  }

  .item {
  grid-column: 1 / 3; /* chiếm 2 cột đầu tiên */
  grid-row: 2 / 4; /* chiếm 2 hàng thứ 2 và thứ 3 */
  }
Trong ví dụ này, phần tử có lớp .item được xác định sử dụng thuộc tính grid-column và grid-row để xác định vị trí và kích thước của nó trong lưới.

Sử dụng grid-column và grid-row để xác định vị trí và kích thước của phần tử trong lưới: 

  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(4, 100px);
  }
  
  .item-1 {
    grid-column: 1 / 3; /* chiếm 2 cột đầu tiên */
    grid-row: 1; /* chiếm hàng đầu tiên */
  }
  
  .item-2 {
    grid-column: 3; /* chiếm cột cuối cùng */
    grid-row: 2 / 4; /* chiếm 2 hàng thứ 2 và thứ 3 */
  }

Sử dụng giá trị auto để đặt kích thước của phần tử dựa trên nội dung bên trong nó: 

  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, auto);
  }
  
  .item-1 {
    grid-column: 1; /* chiếm cột đầu tiên */
    grid-row: 1; /* chiếm hàng đầu tiên */
  }
  
  .item-2 {
    grid-column: 2 / 4; /* chiếm 2 cột thứ 2 và thứ 3 */
    grid-row: 1; /* chiếm hàng đầu tiên */
  }
  
  .item-3 {
    grid-column: 1; /* chiếm cột đầu tiên */
    grid-row: 2; /* chiếm hàng thứ 2 */
  }
  
  .item-4 {
    grid-column: 2; /* chiếm cột thứ 2 */
    grid-row: 2; /* chiếm hàng thứ 2 */
  }
Trong ví dụ này, lưới có 2 hàng được tự động điều chỉnh kích thước dựa trên nội dung của phần tử. Các phần tử được xác định sử dụng grid-column và grid-row, và được đặt kích thước dựa trên nội dung của chúng.

Sử dụng giá trị min-content để đặt kích thước của phần tử dựa trên nội dung nhỏ nhất có thể: 

  .container {
    display: grid;
    grid-template-columns: repeat(3, min-content);
  }
  
  .item-1 {
    grid-column: 1; /* chiếm cột đầu tiên */
  }
  
  .item-2 {
    grid-column: 2; /* chiếm cột thứ 2 */
  }
  
  .item-3 {
    grid-column: 3; /* chiếm cột cuối cùng */
  }

Sử dụng grid-gap để tạo khoảng cách giữa các phần tử trong lưới: 

  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(4, 100px);
    grid-gap: 20px;
  }
  
  .item {
    background-color: #ccc;
    padding: 20px;
    text-align: center;
  }
Trong ví dụ này, grid-gap được sử dụng để tạo khoảng cách 20px giữa các phần tử trong lưới.

Sử dụng grid-row-gap để tạo khoảng cách giữa các hàng trong lưới: 

  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(4, 100px);
    grid-row-gap: 20px;
  }
  
  .item {
    background-color: #ccc;
    padding: 20px;
    text-align: center;
  }
Trong ví dụ này, grid-row-gap được sử dụng để tạo khoảng cách 20px giữa các hàng trong lưới. Chú ý rằng khoảng cách giữa các cột sẽ không bị thay đổi.

Sử dụng grid-area để đặt các phần tử vào vị trí cụ thể trong lưới: 

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(4, 100px);
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "sidebar main main"
    "footer footer footer";
}

.item {
  background-color: #ccc;
  padding: 20px;
  text-align: center;
}

.header {
  grid-area: header;
}

.sidebar {
  grid-area: sidebar;
}

.main {
  grid-area: main;
}

.footer {
  grid-area: footer;
}
Trong ví dụ này, chúng ta đã sử dụng grid-template-areas để đặt tên cho các khu vực trong lưới. Sau đó, chúng ta đã sử dụng grid-area để đặt các phần tử vào các vị trí cụ thể trong lưới.

Sử dụng named grid để đặt tên cho các hàng và cột trong lưới: 

.container {
  display: grid;
  grid-template-columns: [left] 100px [center] 1fr [right] 100px;
  grid-template-rows: [top] 100px [middle] 1fr [bottom] 100px;
}

.item {
  background-color: #ccc;
  padding: 20px;
  text-align: center;
}

.item1 {
  grid-column: left / center;
  grid-row: top / middle;
}

.item2 {
  grid-column: center / right;
  grid-row: top / middle;
}

.item3 {
  grid-column: left / right;
  grid-row: middle / bottom;
}
Trong ví dụ này, chúng ta đã sử dụng named grid để đặt tên cho các hàng và cột trong lưới. Sau đó, chúng ta đã sử dụng các tên này để đặt vị trí cho các phần tử trong lưới bằng cách sử dụng grid-column và grid-row.
