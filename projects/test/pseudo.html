Pseudo-elements are a type of selector in CSS that allow you to target a specific part of an element and style it separately from the rest of the element. Pseudo-elements are identified by two colons (::) followed by a keyword that represents the part of the element you want to target. Here are some common examples of pseudo-elements in CSS:

  • ::before - targets the content before an element, which can be used to add decorative elements such as icons or borders
  • ::after - targets the content after an element, which can be used to add decorative elements such as icons or text
  • ::first-letter - targets the first letter of a block of text, which can be used to apply special styles such as a drop cap
  • ::first-line - targets the first line of a block of text, which can be used to apply special styles such as a different font or color
Pseudo-elements can be very useful for adding decorative or structural elements to web pages without the need for additional HTML markup. For example, you can use the ::before and ::after pseudo-elements to create custom bullet points for a list, or the ::first-letter pseudo-element to create a special effect for the first letter of a paragraph.

You can use the ::first-letter pseudo-element to add a special effect to the first character of a text!
