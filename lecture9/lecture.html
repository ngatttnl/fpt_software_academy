Table

Updating

Set the background color of the table headers and rows:

    th {
      background-color: #ccc;
    }
    tr:nth-child(even) {
      background-color: #f2f2f2;
    }

Add a hover effect to the table rows:

    tr:hover {
      background-color: #ddd;
    }

Change the font family and size of the table text:

    table {
      font-family: Arial, sans-serif;
      font-size: 14px;
    }

Add padding to the table cells:

    td, th {
      padding: 8px;
    }

Set the border of the table and its cells:

    table {
      border-collapse: collapse;
      border: 1px solid #ddd;
    }
    td, th {
      border: 1px solid #ddd;
    }

Style the header row differently from the other rows:

    th {
      background-color: #ccc;
      color: #fff;
      font-weight: bold;
      text-align: left;
    }

Center the table within its container:

      table {
      margin: 0 auto;
    }

Change the background color of alternate rows:

      tr:nth-child(even) {
      background-color: #f2f2f2;
}

Đổi màu nền cho dòng đầu tiên và dòng cuối cùng:

tr:first-child { background-color: #4CAF50; color: white; } tr:last-child { background-color: #4CAF50; color: white; }
